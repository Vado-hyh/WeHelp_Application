Question 6: 關於這份申請網頁，分享一個開發時的技術心得。

在製作這份申請網頁時，我首先使用 Figma 製作了 mockup，可以清楚的知道有哪些部分需要製作，而不會陷入不知從何開始，也知道在製作上有幾個點需要完成以及可能的挑戰。例如，我希望 Navbar 不僅能提供基礎的頁面跳轉功能，還需要實現以下特性：

  1. 滑鼠懸停時圖標顏色改變
    在設計階段，我選擇了像素風的圖標配合像素風字體以強化整體風格，並希望在互動中通過顏色變化提升視覺效果。起初我嘗試使用外部 SVG 文件，但發現無法動態改變顏色。後來，我學習了如何將外部 SVG 文件改為內聯 SVG，並通過 CSS 的 fill 屬性實現顏色的動態切換，最終達成了預期效果。
  2. Navbar 置頂
    在設計時，我希望提升用戶體驗，讓 Navbar 始終可見，避免用戶需要返回頁面頂部才能操作。開發過程中，我研究了 CSS 的 position: sticky 和 position: fixed 屬性，並測試了不同瀏覽器和裝置的顯示效果，成功讓 Navbar 在滾動時保持穩定置頂。
  3. 下拉選單的實現
    在設計 mockup 時，我規劃可點選「Question List」下拉選擇題目。這部分詢問chatGPT說是需要配合 JavaScript 處理互動效果，透過他的協助了解， 通過設計hover 和 click 事件觸發的樣式切換，讓選單能流暢地展開和收起。

這次製作 Navbar 的經驗，不僅讓我從設計到開發的流程更加清晰，也讓我在解決圖標互動、固定定位和響應式設計方面有了更深入的理解。也了解到想要將設計稿用程式開發出來這之間的不易，也會讓我在做設計稿時多一層考慮技術上的可行性。

Question 7: 如何看待自身工作和整個社會群體的連結關係？

我認為工作不僅是展現個人專業能力的地方，還是貢獻社會和改善生活品質的一部分。以前端工程師為例，我們不僅在編碼和設計上發揮專長，更透過每一個細節影響到使用者的數位體驗。優秀的數位產品能夠幫助人們更高效地獲取資訊，提升工作與生活的效率，並且更能包容不同需求的使用者，尤其是對於老年人或有特殊需求的群體，數位服務的可及性和便利性能帶來實質的改變。

當我看到自己設計的網站或應用程式能幫助到真實的人，無論是讓他們輕鬆找到所需資訊，還是讓他們的生活變得更加便捷，我就會覺得這份工作不僅有價值，還能為社會帶來正向影響。這也是我對工作充滿熱情的原因，我相信，作為一名設計師或工程師，我能夠在這個不斷變化的世界中，利用自己的專業能力，為更多人創造更好的數位體驗，並且在過程中實現自我價值。
